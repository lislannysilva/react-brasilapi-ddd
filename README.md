# React BrasilAPI DDD

Aplicação web desenvolvida com React + Vite para consulta de informações de DDD de telefone fixo utilizando a API pública BrasilAPI.

- Vite
- JavaScript

- API REST (BrasilAPI)

- Consumo de API externa
- Tratamento de erros
- Interface responsiva
